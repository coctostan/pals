### Pi Extension API Quick Reference
- Extensions: `export default function(pi: ExtensionAPI) { ... }`
- Discovery: `~/.pi/agent/extensions/` (global), `.pi/extensions/` (project-local)
- Events: `session_start`, `before_agent_start`, `agent_start`, `agent_end`, `turn_start`, `turn_end`, `tool_call` (can block), `tool_result` (can modify), `context` (modify messages)
- `pi.registerCommand("name", { description, handler })` for slash commands
- `pi.registerTool()` for custom tools with TypeBox schemas
- `pi.appendEntry()` for persistent state across restarts
- `ctx.ui.notify()` for plain text notifications
