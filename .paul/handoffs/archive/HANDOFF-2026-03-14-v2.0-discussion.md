### Pi Extension Key Facts
- Extensions are TypeScript modules exporting `default function(pi: ExtensionAPI)`
- Events: `session_start`, `before_agent_start`, `tool_call` (can block), `tool_result` (can modify), `agent_end`, `context` (modify messages)
- Custom tools: `pi.registerTool()` with TypeBox schemas
- Custom commands: `pi.registerCommand()` → `/command-name`
- Pi can load Claude Code skills via settings: `{ "skills": ["~/.claude/skills"] }`
- No `@` file reference syntax — LLM must use read tool for file access
