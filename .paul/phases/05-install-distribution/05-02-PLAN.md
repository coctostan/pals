## AC-1: Uninstall Removes All PALS Files
```gherkin
Given PALS is installed (install.sh has been run)
When uninstall.sh is run
Then ~/.pals/ is removed entirely
And ~/.claude/commands/pals/ is removed
And ~/.claude/commands/carl/ is removed
And ~/.claude/hooks/carl-hook.py symlink is removed
And the CARL hook entry is removed from ~/.claude/settings.json
And no other ~/.claude/ files are affected
```

## AC-2: Uninstall Is Safe
```gherkin
Given PALS is not installed (nothing at ~/.pals/)
When uninstall.sh is run
Then no errors occur
And script reports "PALS not installed, nothing to remove"
```

## AC-3: README Covers Getting Started

```gherkin
And they see install instructions (git clone + ./install.sh)

And they see how to uninstall (./uninstall.sh)
```

<task type="auto">
  <name>Task 1: Create uninstall.sh</name>
  <files>uninstall.sh</files>
  <action>
    Create uninstall.sh at repo root with these sections:

    1. **Header:**
       - #!/bin/bash with set -e
       - Confirm PALS is installed (check ~/.pals/ exists)
       - If not installed: print message and exit 0 (not error)

    2. **Remove framework files:**
       - rm -rf ~/.pals/

    3. **Remove commands:**
       - rm -rf ~/.claude/commands/pals/
       - rm -rf ~/.claude/commands/carl/

    4. **Remove hook:**
       - rm -f ~/.claude/hooks/carl-hook.py

    5. **Deregister hook from settings.json:**
       - Use python3 to remove the carl-hook entry from UserPromptSubmit array
       - If UserPromptSubmit becomes empty, remove the key
       - If hooks becomes empty, remove the key
       - Preserve all other settings.json content

    6. **Summary:**
       - Print what was removed
       - Print "PALS uninstalled successfully"

    Make executable with chmod +x.
    Avoid: Removing any files outside ~/.pals/ and ~/.claude/ (commands + hooks + settings only).
    Avoid: Touching the user's ~/.claude/CLAUDE.md or other non-PALS files.
  </action>
  <verify>
    - uninstall.sh exists and is executable
    - bash -n uninstall.sh passes (syntax check)
  </verify>
  <done>AC-1, AC-2 satisfied: Uninstall removes all PALS files safely</done>
</task>
