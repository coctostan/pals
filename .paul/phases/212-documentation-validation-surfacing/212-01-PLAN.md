- `drivers/pi/skill-map.md` maps workflow/resource capsules as adapter-side context aids over installed resources, not skills or workflow authority.
- Pi and cross-harness validation scripts add focused marker checks for capsule schema, installed/logical citations, fallback, exclusions, validation-class markers, command-output truth, and non-authority boundaries.
