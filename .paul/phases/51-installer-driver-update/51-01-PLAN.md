<objective>
## Goal
Update the Pi driver installer to deploy SKILL.md files and the pals-hooks.ts extension, and update driver.yaml to reflect new capabilities (command registration and hook registration).

## Purpose
Phase 49 created 11 SKILL.md files and Phase 50 created the pals-hooks.ts extension, but the installer doesn't know about them. Without this update, `./install.sh` deploys kernel+module files but skips skills and extensions — making PALS non-functional on Pi.

## Output
- Updated `drivers/pi/install.sh` — deploys skills + extension
- Updated `drivers/pi/uninstall.sh` — cleans up skills + extension
- Updated `drivers/pi/driver.yaml` — capabilities reflect reality
</objective>
