<objective>
## Goal
Pare Pi PALS skill wrapper/default reply guidance so skills are concise by default, route to exactly one next action where applicable, and expand only when blockers, evidence, checkpoints, or user requests require detail.

## Purpose

Phase 231 established contextual verbosity rules for workflows; this plan applies those rules to the Pi skill wrappers without moving workflow authority into skills.

## Output
Updated `drivers/pi/skills/*/SKILL.md` wrappers that preserve canonical workflow links and guardrails while reducing routine reply verbosity, especially for `paul-config`, `paul-help`, and `paul-review`.
</objective>
