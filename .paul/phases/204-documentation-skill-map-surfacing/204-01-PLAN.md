## Goal
Surface the hardened artifact-slice contract in user-facing Pi documentation and the Pi skill map so the implemented slices are clearly described as orientation aids, not lifecycle authority or full-read replacements.

## Purpose
Phase 203 hardened the repo-source Pi runtime slice output for `current-lifecycle-state`, `active-roadmap-phase`, and `approved-plan-task-packet`. - `drivers/pi/skill-map.md` maps artifact-slice loading into the command/skill/workflow surface without making it a skill, workflow, hidden state, or lifecycle authority.
