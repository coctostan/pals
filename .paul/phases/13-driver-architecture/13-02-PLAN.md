<objective>
## Goal
Create Agent SDK driver stub under `drivers/agent-sdk/` that proves the PALS kernel can run headless — same plan/apply/unify loop, no slash commands, driven by the `claude-agent-sdk` Python package.

## Purpose
Proves the "universal drivers" promise of the architecture. If the same workflow markdown can drive both Claude Code (via slash commands) and the Agent SDK (via system prompts), the kernel is truly platform-agnostic.

## Output
- `drivers/agent-sdk/driver.yaml` — driver capabilities manifest
- `drivers/agent-sdk/pals-agent.py` — stub orchestrator using `query()` API
- `drivers/agent-sdk/requirements.txt` — dependencies
- `drivers/agent-sdk/README.md` — setup and usage guide
</objective>
