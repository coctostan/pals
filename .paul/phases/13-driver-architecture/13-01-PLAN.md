<objective>
## Goal
Extract Claude Code driver from current install.sh into `drivers/claude-code/` directory. Refactor root install.sh to install kernel files, then delegate to the active driver's installer for platform-specific setup.

## Purpose
Separates platform-specific (Claude Code) logic from platform-agnostic kernel install. This is the foundation for multi-driver support — any future driver just adds a directory under `drivers/` with its own install.sh.

## Output
- `drivers/claude-code/driver.yaml` — driver capabilities manifest
- `drivers/claude-code/install.sh` — Claude Code platform setup (commands, hooks, settings.json)
- `drivers/claude-code/uninstall.sh` — Claude Code platform teardown
- Root `install.sh` refactored: kernel install + driver delegation
- Root `uninstall.sh` refactored: driver teardown + kernel cleanup
</objective>
