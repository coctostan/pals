# Plan 246-01 — Extract S2 `workflow-resource-capsule-rendering` to a Sibling Pi Extension Module

<objective>

## Goal

Extract the S2 `workflow-resource-capsule-rendering` subsystem from `drivers/pi/extensions/pals-hooks.ts` into a new sibling Pi extension module `drivers/pi/extensions/workflow-resource-capsule-rendering.ts`, keeping `pals-hooks.ts` as imports + a delegated call for that subsystem and preserving every visible capsule marker, payload bound, fallback string, authority string, file-source contract, schema-marker tuple, and call-site behavior. The new sibling MUST end with the canonical Phase 245 no-op default-exported factory so Pi's extension loader does not reject it at session start.

## Purpose

Phase 245 closure recommends "do more of what worked" — repeat the S5 + S1 sibling-module recipe for S2, the second half of the pure/similar pair named in `docs/PI-NATIVE-EXTENSION-EXTRACTION-TARGET-BASELINE.md` (lines 28, 53, 89). Phase 243 SUMMARY already exported the shared helpers (`compactWhitespace`, `readFileOr`, `getFileFreshness`, `selectBoundedLines`, `escapeRegExp`, type `PalsStateSnapshot`) precisely so future S2-S8 extractions can import them from `./pals-hooks`. The Phase 241 generalized install surface auto-covers any new repo-owned `drivers/pi/extensions/*.ts` file by basename, so this plan needs no installer, `driver.yaml`, manifest, kernel, modules, contract docs, `tests/helpers/**`, or `tests/cross-harness-validation.sh` edits to deploy. **Critical Phase 245 invariant:** Pi auto-loads every `~/.pi/agent/extensions/*.ts` and rejects modules without a default-exported factory; the new sibling MUST end with `export default function (_pi: unknown): void {}` preceded by a JSDoc comment carrying the marker phrase `No-op Pi extension factory` (mirroring `module-activity-parsing.ts` and `artifact-slice-rendering.ts`).

## Output

Shared helpers (`compactWhitespace`, `readFileOr`, `getFileFreshness`, `selectBoundedLines`, `escapeRegExp`, type `PalsStateSnapshot`) remain defined and exported in `pals-hooks.ts`.
