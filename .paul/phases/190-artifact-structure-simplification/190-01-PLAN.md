## Goal
Apply the validated Phase 185 STATE archive pattern to `.paul/ROADMAP.md`: move completed-milestone history out of the hot lifecycle artifact into a new indexed archive, leaving an active-window roadmap with Overview + Milestones index + Current Milestone inline.

## Purpose

After this plan, every PLAN/APPLY/UNIFY/resume/status turn that consults the roadmap reads ~92% fewer lines while completed milestone history remains authoritative and findable through the archive.

## Output
- A new archive file at `.paul/archive/roadmap/ROADMAP-HISTORY-v0-v2.43.md` holding the 11 completed-milestone `<details>` blocks currently inline in ROADMAP.md.
- A compacted `.paul/ROADMAP.md` (target ≤ 100 lines) containing Overview + Milestones index + Current Milestone + a single inline pointer to the new archive.
