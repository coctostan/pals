## Goal
Extract the 7 pure/derived helper functions + the `PalsStateSnapshot` type from `drivers/pi/extensions/pals-hooks.ts` into a new dependency-leaf sibling `drivers/pi/extensions/shared-runtime-helpers.ts`, repoint all 9 importing files (pals-hooks.ts + 8 siblings) to consume the moved symbols from the new leaf, remove the now-duplicate definitions from pals-hooks.ts, and repoint the Pi-e2e TAP assertions that pin the old import locations — with no import cycles, no new authority markers, and no net behavior change.

Executes against `docs/PALS-SHARED-RUNTIME-HELPERS-CONTRACT.md` (Phase 303, PR #223).

- 9 repointed importers (pals-hooks.ts + 8 siblings).
- Updated `tests/pi-end-to-end-validation.sh` TAP assertions for the new import locations + a new bounded leaf-extraction TAP block.
