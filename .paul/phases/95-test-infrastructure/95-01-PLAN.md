<task type="auto">
  <name>Task 3: Launch Pi with kimi-coder model and verify PALS</name>
  <files>(no source files)</files>
  <action>
    1. In the test pane, cd to project: `cmux send --surface <ref> "cd /Users/maxwellnewman/claude/pi-verify"`
    2. Launch Pi with model: `cmux send --surface <ref> "pi --model 'kimi-coder/kimi-for-coding'"`
    3. Wait 5 seconds for Pi to start
    4. Read screen: `cmux read-screen --surface <ref> --lines 20`
    5. Verify Pi started (look for assistant greeting or prompt)
    6. Send PALS test: `cmux send --surface <ref> "/paul-status"`
    7. Wait 3 seconds
    8. Read screen and verify PALS responds (not "command not found")
  </action>
  <verify>

    - /paul-status command is recognized by PALS (shows project status or "no project found")
  </verify>
