## Key Decisions

- **Contract-first, source-verified (Phase 303):** The contract documented the ACTUAL fuller co-import sets, not the PLAN narrative's abbreviated list (`lifecycle-ui.ts` also imports `MAX_VISIBLE_MODULES` + `RECENT_MODULE_ACTIVITY_LOOKBACK`; `command-routing.ts` and `pals-context-injection.ts` carry `ActivationState` as type-only). "Correct the contract, not the source" governs.
- **Leaf is fully Node-only; no type-only back-import (Phase 304):** `PalsStateSnapshot` moves WITH the helpers, so the leaf is a genuine sink — cleaner than S6/S7/S8 which needed type-only back-imports.
- **Split imports preserve stay-put membership (Phase 304):** `command-routing.ts` and `pals-context-injection.ts` split their imports — `ActivationState` stays on `./pals-hooks`, `PalsStateSnapshot` moves to the leaf — preserving the Phase 303 contract's stay-put set exactly.
